AIFaceRecognition App (prototype)
----------------------------
This project sets up a simple test to work with Clarifai's face recognition AI.

To Run
----------------------------
npm install
npm start

Working Example
----------------------------
Open index.html in the dist folder or go to http://dominicknorwood.com/faceai/.
